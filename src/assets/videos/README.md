# Videos

Place your green screen videos here.

## Supported Formats

- MP4 (.mp4) - Recommended
- WebM (.webm)
- MOV (.mov)
- MKV (.mkv)
- AVI (.avi)

## Requirements

- Green screen background (default: #00ff00)
- Maximum file size: 500MB per video
- Recommended resolution: 480p - 1080p
- Recommended frame rate: 24-60 fps

## Tips

1. Use consistent green background color for best results
2. Good lighting on the green screen helps with keying
3. Avoid shadows on the green screen
4. Test different similarity/smoothness values in settings
